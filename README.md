## Giới thiệu tổng quan
Trong bối cảnh lĩnh vực thương mại điện tử đang có xu hướng phát triển mạnh mẽ , nhóm nhận thấy ngành hàng chăm sóc bản thân và làm đẹp, đặc biệt là sản phẩm skincare, vẫn còn thiếu vắng những nền tảng online chuyên biệt và tối ưu.

Đó là lý do nhóm xây dựng **Derma.glow** — một website tập trung hoàn toàn vào sản phẩm skincare, được thiết kế để giải quyết nhu cầu giúp người dùng tiếp cận sản phẩm một cách dễ dàng và hiện đại.

## Công nghệ sử dụng
* **HTML**: Ngôn ngữ đánh dấu tiêu chuẩn để xây dựng cấu trúc về mặt nội dung cho trang web.
* **CSS**: Dùng để thiết kế trang web.
* **JavaScript**: Ngôn ngữ đóng vai trò tạo ra các trang web động và có tương tác.
## Cấu trúc thư mục
```
├── accounts         # Thư mục chứa các tệp HTML liên quan đến quản lý tài khoản (Đăng nhập, Đăng ký, Cài đặt tài khoản, Lịch sử mua hàng, Sản phẩm yêu thích) 
├── cart             # Thư mục trang giỏ hàng và chi tiết sản phẩm 
├── components       # Các thành phần giao diện có chức năng riêng biệt, độc lập và dùng chung cho tất cả các trang (header, footer)
├── css              # Chứa tất cả các tệp CSS cho tất cả các trang và components 
├── homepage         # Thư mục trang chủ 
├── js               # Chứa tất cả các tệp JavaScript cho tất cả các trang và components 
├── products         # Các trang danh sách sản phẩm
└── index.html       # Trang khởi chạy chính điều hướng đến trang chủ của website
